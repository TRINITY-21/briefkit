Endpoints

All endpoints require an API key via X-API-Key header.

POST /api/keys Generate a new API key
GET /api/weather/{city} Weather data for any city
GET /api/crypto/{coins} Crypto prices, 24h change, market cap
GET /api/news?limit=5 Top Hacker News stories
GET /api/briefing?city=Istanbul Combined weather + crypto + news

Quick Start

Get your API key and start making requests in seconds.

1. Get your API key bash 
# Generate a free API key
curl -X POST /api/keys

# Response:
{
  "data": {
    "key": "bk_fa77555889ce4557..."
  }
}
2. Make requests bash 
# Get a full briefing
curl -H "X-API-Key: bk_your_key" \
  /api/briefing?city=Istanbul

# Get crypto prices
curl -H "X-API-Key: bk_your_key" \
  /api/crypto/bitcoin,ethereum

Response Format

All responses follow a consistent JSON structure.

GET /api/weather/Istanbul 200 OK 
{
  "status": "ok",
  "data": {
    "city": "Istanbul",

Features

Built for developers who need reliable data fast.

🔑

API Key Auth

Generate keys instantly. Pass via X-API-Key header.

⚡

TTL Cache

In-memory caching prevents upstream API hammering. Fast responses.

🛡

Rate Limiting

100 requests/day per key. Fair usage for everyone.

📖

Auto Docs

Interactive Swagger UI with typed schemas at /docs.

🌐

CORS Enabled

Call from any browser app. All origins allowed.

🚀

Parallel Fetches

/briefing fetches all 3 sources concurrently via asyncio.
